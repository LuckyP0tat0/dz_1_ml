# Датасет выбран по следующим причинам:
— задача регрессии (предсказание цены), ранее не работал с регрессией;
— присутствуют как числовые, так и категориальные признаки;
— прикладная и понятная бизнес-задача;
— интересно проверить влияние регуляризации (Ridge и Lasso) на качество модели.

# Список гипотез (до начала работы)

1. Возраст автомобиля существенно влияет на цену: чем старше автомобиль, тем ниже его стоимость.
2. Пробег обратно пропорционален цене: увеличение mileage снижает стоимость.
3. Наличие аварий снижает стоимость автомобиля.
4. Регуляризация улучшит обобщающую способность модели и уменьшит переобучение.
5. Ridge будет работать лучше Lasso, если признаки коррелированы и важны совместно.
6. Наличие выбросов увеличивает RMSE сильнее, чем MAE, так как RMSE квадратично штрафует большие ошибки.

# Краткие результаты

1. Гипотеза о влиянии года выпуска подтвердилась: более новые автомобили стоят дороже.
2. Гипотеза о пробеге подтвердилась: рост пробега снижает цену.
3. Наличие аварий оказывает влияние на стоимость, однако эффект не во всех случаях критичен.
4. Регуляризация улучшила стабильность модели на кросс-валидации.
5. Ridge показал более устойчивый результат по R² по сравнению с Lasso.
6. Гипотеза подтвердилась: после удаления выбросов абсолютные ошибки (MAE и особенно RMSE) существенно снизились, что говорит о сильном влиянии экстремальных значений на модель.

Модели настраивались с использованием GridSearchCV (5-fold cross-validation).
Оценка проводилась по метрикам R², MAE и RMSE.
